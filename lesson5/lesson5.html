Теги b и strong используются чтобы сделать визуально текст насыщеннее.

Основное отличие между тегами b и strong заключается в их смысловой нагрузке: strong указывает на семантическую важность текста (логическое ударение), тогда как b предназначен для визуального выделения текста без придания ему особого смысла, например, для названий или ключевых слов. Хотя в браузере оба тега по умолчанию отображают текст полужирным шрифтом, strong лучше использовать для усиления значимости, а b — для стилистических отличий от нормального текста.

Тег i используется для стилизации текста курсивом без смысловой нагрузки, например, для иностранных слов или названий. Тег em (emphasis) обозначает семантический акцент на тексте, указывая на его особую важность в предложении, и также отображается курсивом. Главное отличие — em несет смысловую нагрузку, а i служит только для оформления.

Тег "s" используется для определения текста, который больше не актуален. Содержимое тега браузеры отображают как перечеркнутый текст.

В тег "u" заключается текст, который стилистически отличается от остального текста. Это могут быть слова с орфографическими ошибками, слова, которые должны быть подчеркнуты по правилам языка. Специально допустим ашибку в этом предложении.

тег "sub" используется для определения текста с нижним индексом. Текст в нижнем индексе располагается ниже базовой линии остальных символов строки и отображается уменьшенным шрифтом. Текст с нижним индексом чаще всего используется для написания формул. Формула воды -H2O, а формула спирта - C2H5ОН

Тег "sup" используется для определения текста в верхнем индексе, который располагается выше базовых линий остальных символов строки и отображается уменьшенным шрифтом. Текст в верхнем индексе чаще всего используется для написания сносок или формул. (a+b)2=a2+2ab+b2

E = mc2, где E — энергия объекта, m — его масса, c — скорость света в вакууме.

Тег "small" уменьшает размер вложенного в него текста на одну единицу, кроме тех случаев, когда вложенный текст написан наименьшим шрифтом. Тег "small" содержит дополнения к основному тексту. Таковыми могут являться сноски, комментарии, уведомления об авторских правах, какие-либо ограничения или цитаты. Содержимое тега по умолчанию отображается меньшим размером, чем окружающий текст. Теги "small" могут вкладываться один в другой, при этом размер текста будет уменьшаться пропорционально уровню вложенности.

Тег "mark" выделяет части текста, с которыми работает пользователь в данный момент, и которые представляют для него важность. Визуально содержимое тега выглядит как отмеченное маркером. Тег "mark" можно использовать, например, для выделения результатов поиска на странице, либо для того, чтобы подсветить текущую дату в календаре. В браузерах Chrome и Firefox содержимое тега выделяется желтым фоном, однако это можно изменить при помощи CSS свойства background-color.

Тег "br" устанавливает перевод строки, то есть все написанное после него, будет перенесено на новую строку. В отличие от тега "p", определяющего абзац, перед строкой не добавляется пустой отступ.

Пример использования тега <br>

Внутри абзаца мы можем вставить тег <br />,
чтобы перенести часть текста на другую строку в случае необходимости.

В HTML 5 тег "hr" используется для тематического разделения контента на странице. В предыдущих версиях HTML он использовался для создания горизонтальной линии на странице, визуально разделяя контент. В новой версии он приобрел смысловую нагрузку и определяет не только визуальный, но и тематический разрыв контента.

Тег "code" используется для вставки в HTML документ строк программного кода, текстов функций, переменных и т.д. В браузере код отображается моноширинным шрифтом (шрифт, в котором все символы одной ширины) меньшего размера.

Каждый HTML-файл начинается со строчки кода с указанием doctype

Тег "pre" используется для включения в HTML-документ предварительно отформатированного текста. Во вложенном в тег тексте сохраняются все пробелы и разрывы строк (как известно, браузеры по умолчанию любое количество идущих подряд пробелов показывают как один). Тег "pre" используют для отображения программного кода, либо текста (например, стихотворения), где автор сам задал расположение строк относительно друг друга.
